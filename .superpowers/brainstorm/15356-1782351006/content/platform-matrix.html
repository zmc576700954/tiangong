关键决策说明

Windows ARM64 + libsql：通过 x64 模拟层运行，Windows 11 ARM64 原生支持 x64 进程模拟。安装包标注"通过 x64 兼容层运行数据库模块"，用户无感知。

MSIX 格式：支持 x64 + arm64 双架构打包（MSIX 可在一个包内包含多架构二进制），分发到 Microsoft Store 时必需。

macOS：维持当前 arm64 + x64 双 DMG 策略，不做 Universal Binary（体积大、构建慢）。
